ما هي HTML؟

HTML اختصار لـ HyperText Markup Language، وتعني "لغة ترميز النص التشعبي". وهي اللبنة الأساسية لإنشاء صفحات الويب. باستخدام HTML يمكنك تحديد العناوين، الفقرات، الروابط، الصور، القوائم، الجداول، والعديد من العناصر الأخرى التي تُكوّن محتوى الموقع.

لماذا تعتبر HTML مهمة؟

HTML تُعدّ الأساس الذي تُبنى عليه مواقع الإنترنت. بدونها، لا يمكن للمتصفح عرض أي محتوى منظم. تُستخدم HTML في كل صفحة ويب — حتى التطبيقات الحديثة المعتمدة على JavaScript أو React تبدأ بـ HTML.

بنية صفحة HTML الأساسية

  • <!DOCTYPE html>: يُخبر المتصفح أن هذه الصفحة تستخدم HTML5.
  • <html>: الجذر الذي يحتوي على كل العناصر الأخرى.
  • <head>: يحتوي على البيانات التعريفية مثل العنوان والروابط الخارجية.
  • <body>: يحتوي على كل ما يظهر للمستخدم مثل النصوص، الصور، والأزرار.

أهم الوسوم في HTML

  • <h1> - <h6>: لكتابة العناوين بمستويات مختلفة.
  • <p>: لكتابة الفقرات.
  • <a href="...">: لإنشاء الروابط.
  • <img src="...">: لإضافة الصور.
  • <ul> و <ol>: لإنشاء القوائم.
  • <div> و <span>: لتقسيم المحتوى.

هل HTML لغة برمجة؟

HTML ليست لغة برمجة لأنها لا تحتوي على منطق برمجي مثل الشروط أو الحلقات. بل تُستخدم لتوصيف هيكل المحتوى فقط، وتُكملها CSS لتنسيق الشكل، وJavaScript لإضافة التفاعلية.

فوائد تعلم HTML

  • البداية المثالية لتعلم تطوير الويب.
  • تساعدك على فهم بنية أي موقع إلكتروني.
  • تمكنك من إنشاء مشاريعك الشخصية بسرعة.
  • أساسية عند تعلم أي إطار عمل مثل React أو Vue.

الخطوة التالية

بعد تعلم HTML، يوصى بتعلم CSS لتصميم الصفحات، ثم JavaScript لجعل الموقع تفاعلياً. يمكنك بعد ذلك تعلم مكتبات وأطر عمل حديثة مثل Bootstrap أو React.
