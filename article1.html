مقال: مقدمة إلى HTML

ما هي HTML؟

HTML هي اختصار لـ HyperText Markup Language، أي "لغة ترميز النص التشعبي"، وهي اللغة الأساسية المستخدمة لإنشاء صفحات الويب. تُستخدم HTML لتحديد الهيكل والمحتوى الأساسي لأي صفحة، مثل العناوين، الفقرات، الروابط، الصور، الجداول والقوائم.

لماذا HTML مهمة؟

بدون HTML، لا يمكن عرض المحتوى على الإنترنت بطريقة منظمة ومفهومة للمتصفحات. فهي بمثابة العمود الفقري لأي صفحة ويب. حتى عند استخدام لغات أخرى مثل CSS أو JavaScript، فإنها تعتمد على HTML كأساس.

بنية صفحة HTML

تتكون أي صفحة HTML من العناصر التالية:

  • <!DOCTYPE html>: تُخبر المتصفح بأن هذه الصفحة مكتوبة بلغة HTML5.
  • <html>: العنصر الجذري الذي يحتوي على باقي عناصر الصفحة.
  • <head>: يحتوي على بيانات غير مرئية للمستخدم مثل العنوان، الروابط، التعريفات.
  • <body>: يحتوي على المحتوى المرئي مثل النصوص، الصور، الأزرار، إلخ.

أشهر الوسوم في HTML

  • <h1> إلى <h6>: لكتابة العناوين.
  • <p>: لكتابة الفقرات.
  • <a href="">: لإنشاء روابط.
  • <img src="">: لإدراج الصور.
  • <ul> و <li>: للقوائم.
  • <div>: لتقسيم الصفحة إلى أقسام قابلة للتصميم.

هل HTML لغة برمجة؟

HTML ليست لغة برمجة، بل هي لغة ترميز أو توصيف. لا تحتوي على منطق برمجي مثل الشروط أو الحلقات، بل تُستخدم فقط لبناء هيكل الصفحة.

ما التالي بعد HTML؟

بعد تعلم HTML، يجب الانتقال لتعلم CSS لتنسيق وتصميم الصفحات، ثم JavaScript لإضافة التفاعلية والديناميكية.

ببساطة، HTML هي البداية المثالية لأي شخص يريد دخول عالم تطوير الويب.
