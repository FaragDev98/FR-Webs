📘 مقدمة في HTML

HTML هي أساس بناء أي صفحة ويب. في هذا الدرس سنتعلم كيفية إنشاء الهيكل الأساسي، إضافة النصوص، الصور، والروابط. بنهاية الدرس ستكون قادرًا على بناء صفحة كاملة بنفسك.

🎨 تنسيق المواقع باستخدام CSS

CSS تضيف الجمال للصفحات. سنتعلم الألوان، الخطوط، المسافات، الظلال، وتأثيرات 3D مذهلة. الدرس طويل ومليء بالأمثلة العملية لتصميم واجهات احترافية.

⚡ التفاعل باستخدام JavaScript

في هذا الدرس سنتعلم كيف نجعل المواقع تفاعلية. سنضيف الأزرار الذكية، الرسائل، والمؤثرات، مع تطبيقات عملية. الدرس متدرج ليأخذك من الصفر إلى الاحتراف.
