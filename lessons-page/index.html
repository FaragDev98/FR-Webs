💻 كورس واجهات المستخدم (HTML & CSS & JS)

تعلم تصميم وتفاعل الويب خطوة بخطوة بأسلوب 3D وتجربة ممتعة

📙 مقدمة في HTML

في هذا الدرس ستتعرف على مفهوم هيكل صفحات الويب وكيفية استخدام الوسوم (Tags) لكتابة المحتوى.

📥 تحميل ملخص HTML

🎨 تنسيق الصفحات باستخدام CSS

تعلم كيف تجعل صفحاتك أنيقة ومتجاوبة مع الشاشات المختلفة باستخدام الألوان، الهوامش، والخطوط.

📥 تحميل ملخص CSS

⚡ التفاعل مع المستخدم بـ JavaScript

ابدأ بفهم كيفية جعل الصفحات حية — مثل التحكم بالأزرار، النصوص، والحركات التفاعلية.

📥 تحميل ملخص JS

🧠 مفاهيم واجهات المستخدم (UI)

واجهة المستخدم هي كل ما يراه ويتفاعل معه المستخدم. تعلم كيف تصمم تجربة بصرية تجعل المستخدم يشعر بالراحة أثناء تصفح موقعك.

🎯 تجربة المستخدم (UX)

هي طريقة تفكير تساعدك على جعل منتجك مفيد وسهل الاستخدام. تعلم كيف تحلل سلوك المستخدم وتصمم وفقًا له.

🧩 الاختبار العملي

قم بإنشاء صفحة ويب تحتوي على:

  • عنوان رئيسي باستخدام HTML
  • تنسيق الخلفية والنصوص بـ CSS
  • زر تفاعلي بـ JavaScript

بعد الانتهاء، احفظ عملك وأرسله للمدرب لتقييمه.
