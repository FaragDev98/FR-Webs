📗 مقدمة HTML

لغة HTML هي أساس أي موقع ويب. من خلالها تبني الهيكل العام للصفحة باستخدام الوسوم. في هذا الدرس سنتعرف على الوسوم الأساسية مثل <html>, <head>, <body>، وكيفية إنشاء النصوص، الصور، والروابط.

📥 تحميل الملخص

🎨 تصميم المواقع باستخدام CSS

CSS تضيف الجمال إلى مواقعك. سنتعلم استخدام الألوان، الخطوط، الخلفيات، والتخطيطات الشبكية لتنسيق الصفحة وجعلها متجاوبة مع جميع الأجهزة. كما سنرى كيفية إنشاء تأثيرات حركية لطيفة باستخدام transition وanimation.

📥 تحميل الملخص

⚡ التفاعل باستخدام JavaScript

JavaScript تجعل الموقع ينبض بالحياة! سنتعرف على طريقة التعامل مع العناصر في الصفحة عبر DOM، وتنفيذ الأوامر بناءً على تفاعل المستخدم (كالنقر والتمرير).

📥 تحميل الأكواد

🧠 مفهوم DOM في JavaScript

الـ DOM هو اختصار لـ Document Object Model وهو الواجهة التي تُمكّن JavaScript من التفاعل مع HTML. بفضله تستطيع تغيير النصوص، الألوان، الصور وحتى إنشاء عناصر جديدة ديناميكيًا في الصفحة.

📄 قراءة الدرس كامل

💡 تنسيق الصفحات باستخدام Flexbox و Grid

تعلم كيفية توزيع العناصر في الصفحة بشكل احترافي باستخدام تقنيات Flexbox وGrid. هذه الأدوات تسهل تصميم صفحات متجاوبة ومنظمة تعمل على الهاتف والكمبيوتر بسهولة.
