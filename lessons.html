📙 درس 1 — HTML: بناء هيكل الموقع (مفصل)

مقدمة: HTML (HyperText Markup Language) هي العمود الفقري لأي صفحة ويب. في هذا الدرس نغطي: هيكل الصفحة، الوسوم الأساسية، السمات (attributes)، كيفية إدراج صور وروابط، والعمل مع النماذج والقوائم والجداول.

أساسيات عملية

ابدأ دائمًا بملف index.html. تذكَّر أهمية وسوم الـ head مثل <meta charset="utf-8"> ووسم عرض الشاشة ليتجاوب الموقع. استخدم وسوم دلالية (<header>, <main>, <footer>, <article>) لتحسين الوصول وSEO.

مثال عملي سريع

<!DOCTYPE html>
<html lang="ar">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width,initial-scale=1">
    <title>صفحتي الشخصية</title>
  </head>
  <body>
    <header><h1>اسمي</h1></header>
    <main><p>نبذة عني...</p></main>
    <footer><p>حقوق النشر</p></footer>
  </body>
</html>

تحميل: يمكنك تحميل قالب HTML جاهز من الرابط أدناه وتعديله مباشرة.

🎨 درس 2 — CSS: تصميم وتزيين الصفحات (مفصل)

CSS تتحكم بالمظهر: الألوان، الخطوط، الفراغات، وتخطيط الصفحة. سنتعلم Box Model، Selectors، Flexbox، Grid، واستراتيجيات تنظيم الأكواد (BEM أو CSS Modules).

مثال Flexbox

.container { display:flex; gap:16px; flex-wrap:wrap; }
.item { flex:1 1 220px; padding:12px; border-radius:8px; background:#fff9; }

نصائح عملية: اختر لوحة ألوان محددة، استعمل خطوط ويب محسّنة، وقلل من أحجام الصور لتحسين الأداء.

⚡ درس 3 — JavaScript: التفاعل وديناميكية الصفحات (مفصل)

JavaScript تضيف التفاعل: التعامل مع DOM، الأحداث، التحكم بالنماذج، والتحقق منها. سنبني وظائف صغيرة: تغيير نصوص، فتح نافذة مودال، والتحكم بالتخزين المحلي (localStorage).

مثال عملي

const btn = document.querySelector('.btn');
btn.addEventListener('click', ()=>{
  document.querySelector('.card p').textContent = 'تم التحديث!';
});

✍️ درس 4 — كيف تبني موقعك الشخصي (Portfolio)

دمجنا HTML/CSS/JS لصناعة موقع شخصي: صفحة رئيسية، معرض أعمال، وصف المهارات، ونموذج اتصال. استخدم قالب الـ ZIP للتعديل السريع.

خطوات موجزة:

  1. أضف ملف index.html وبنية واضحة.
  2. اربط ملف style.css واستخدم Grid لعرض المشاريع.
  3. أضف تفاعل بسيط عبر JS لفتح تفاصيل المشروع في مودال.
